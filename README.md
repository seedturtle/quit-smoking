# 戒菸吧 — 重新開始的勇氣 🍃

戒菸資訊互動網頁，提供：

- 📊 **戒菸省錢計算機** — 即時計算每日／每月／每年可省下的金額
- ⏱️ **戒菸後身體變化時間軸** — 從 20 分鐘到 10 年的健康改善
- 🏥 **台灣戒菸服務資源** — 戒菸門診、戒菸專線（0800-636363）、合約機構查詢
- 💊 **戒菸方法建議** — 藥物治療、諮商輔導、自我管理

### 部署方式

1. 推送到 GitHub
2. 在 [Zeabur](https://zeabur.com) 匯入此專案
3. 設定環境變數 `PORT`（可選，預設 3000）
4. 自動部署

### 技術棧

- 純 HTML + CSS + JavaScript（無框架依賴）
- Node.js 靜態檔案伺服器（Zeabur 部署用）
